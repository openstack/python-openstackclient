=========
user role
=========

Identity v2

user role list
--------------

List user-role assignments

*Removed in version 3.*

.. program:: user role list
.. code:: bash

    os user role list
        [--project <project>]
        [<user>]

.. option:: --project <project>

    Filter users by `<project>` (name or ID)

.. describe:: <user>

    User to list (name or ID)
